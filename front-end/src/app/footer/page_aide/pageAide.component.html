Aide
Bienvenue sur la page d'aide!
Chaque couleur sur le clavier correspond à une action
  • violet : se diriger vers la gauche
  • vert : se diriger vers le haut
  • jaune : se diriger vers le bas
  • bleu : se diriger vers la droite
  • marron : valider votre sélection
  • orange : retour à la page d'avant
  • gris: dirige vers la page d'aide (celle-ci)
Pour toutes autres questions demandez à vos encadrants.
